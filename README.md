# Trove Market Backend

![Backend CI](https://github.com/sadxdev/trove-backend/actions/workflows/ci.yml/badge.svg)

Spring Boot-based backend service for the Trove Market Q-Commerce platform.

## Tech Stack

- Spring Boot 3.5.3
- Java 17
- MySQL 8
- Spring Data JPA

## Setup

```bash
./mvnw spring-boot:run
